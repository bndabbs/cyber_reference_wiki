Notes and Warnings#

NOTE: Notes and Warnings are for key information or critical "you'll shoot your eye out" warnings. Use them sparingly.

This code: 

!!! note
    This is a note

!!! warning
    This is a warning

Creates:

Note

This is a note

Warning

This is a warning
